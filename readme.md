## 使用说明（Windows 系统）

#### 作者: 网云2304 李润东

本软件仅适用于 Windows 操作系统。

### 1. 生成密钥
在进行文件加密或解密之前，您需要先生成一个密钥文件（`.key`）。  
密钥是解密的重要凭证，请妥善保管。

---

### 2. 加密文件
1. 选择需要加密的目标文件。
2. 程序会在相同目录下生成一个后缀为 `.enc` 的加密文件。
3. 加密过程中不会修改原文件。

---

### 3. 解密文件
1. 选择需要解密的 `.enc` 文件。
2. 使用与加密时相同的密钥文件（`.key`）进行解密。  
   如果密钥不匹配，解密将失败。

---

### 4. 日志功能
软件会在以下目录自动创建日志文件：

%USERPROFILE%\AppData\Local\DataLocker\


日志中记录了密钥使用情况和操作时间。  
若忘记文件使用了哪一个密钥，可前往该目录查阅日志。

---

### 5. 重要提示
- 请务必保存好密钥文件，密钥一旦丢失，加密文件将无法恢复。
- 建议不要将密钥文件与加密文件存放在同一文件夹中，以提高安全性。
